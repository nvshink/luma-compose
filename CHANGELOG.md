# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-05-30

- Promoted `luma-compose` to the `1.0.0` release line.
- Expanded the `:luma` module to Kotlin Multiplatform targets for Android, Desktop, iOS, and JS.
- Added the `:desktop-demo` application for desktop verification and iteration.
- Updated CI to validate Android, Desktop, JS, and iOS targets.

## [0.1.0] - 2026-05-25

- First public library release of `luma-compose`.
- Added the `:luma` Android library module with reusable Compose visual effects.
- Added the `:android-demo` demo app with interactive catalog screens for patterns, glow, and combined scenes.
- Added Maven publication metadata, signing hooks, and CI for open-source distribution.
