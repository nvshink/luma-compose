# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-05-25

- First public library release of `luma-compose`.
- Added the `:visuals` Android library module with reusable Compose visual effects.
- Added the `:sample` demo app with interactive catalog screens for patterns, glow, and combined scenes.
- Added Maven publication metadata, signing hooks, and CI for open-source distribution.
